# Escola Conectada

Escola Conectada é uma interface web desenvolvida com **HTML, CSS e JavaScript**.  
O projeto simula um **Sistema de Gestão Escolar** com o objetivo de conectar a instituição de ensino aos pais/encarregados de educação, permitindo a consulta organizada de informações académicas.

Trata-se de um projeto de estudo focado na prática de desenvolvimento front-end, organização estrutural e criação de interfaces institucionais responsivas.

---

## Objetivo do Projeto

O sistema foi desenvolvido para:

- Simular um ambiente real de gestão escolar
- Facilitar a visualização de dados académicos
- Demonstrar a comunicação entre escola e pais
- Aplicar boas práticas de HTML, CSS e JavaScript

---

## Tecnologias Utilizadas

- HTML5 – Estrutura das páginas
- CSS3 – Estilização e layout responsivo
- JavaScript – Interatividade e manipulação do DOM

---

## Funcionalidades

- Interface institucional organizada
- Layout responsivo (desktop e mobile)
- Área simulada de gestão escolar
- Organização de alunos, turmas e notas
- Sistema de login ilustrativo

### Credenciais de Demonstração

Após baixar o projeto:

1. Acesse a tela de login.
2. Clique em **"Esqueceu a senha?"**.
3. Será aberto um **modal** contendo as credenciais de utilizadores demo.
4. Utilize uma das credenciais fornecidas para acessar o sistema.

Essa funcionalidade permite testar o sistema sem necessidade de cadastro ou base de dados.

---

## Estrutura do Projeto

O sistema foi organizado separando:

- Estrutura (HTML)
- Estilização (CSS, Bootstrap 5)
- Lógica e interatividade (JavaScript)

Essa organização facilita manutenção e futura integração com back-end ou base de dados.
